What is Active Record?

Under the hood

3/12/2015

Described as a core part of Ruby on Rails, Active Record is responsible for handling all database interaction from the application level. ActiveRecord maps a database row to an object in Ruby. Here are the key steps Active Record is doing behind the scenes as referenced in this article:

  • It composes an SQL query.
  • It fills in the search parameters.
  • It sends this off to a database API.
  • It runs the query.
  • It examines and parses the result.
  • It then stores the result in either a hash or a database
